OpenAI：GPT-5.4震撼登场，Agent能力全面升级

OpenAI在3月5日发布了GPT-5.4，这是继GPT-5系列之后的又一重大更新。新版本在Agent能力上实现了质的飞跃：

核心亮点

  1. Responses API配备计算机环境 - 3月11日，OpenAI宣布为Responses API配备完整的计算机环境，让Agent能够真正”操作”计算机，而不仅仅是生成文本。这意味着Agent可以自主执行文件操作、运行代码、管理进程等任务。

  2. 提示注入免疫力增强 - 同一天，OpenAI发布了关于优化AI智能体设计的重磅文章，重点阐述了如何提升Agent对”提示注入攻击”的免疫力。这是Agent安全领域的重要突破。

  3. Codex Security预览版上线 - 3月6日，Codex Security研究预览版正式上线，为开发者提供了更安全的代码生成和审计能力。

  4. 收购Promptfoo - 3月9日，OpenAI宣布收购Promptfoo，这是AI提示工程领域的领先公司，此举将大幅增强OpenAI在Agent提示优化方面的能力。

Anthropic：Claude记忆系统引发行业关注

Anthropic在Agent领域的动作同样引人注目：

重大新闻

  • Claude Sonnet 4.6发布 - 2月17日，Claude Sonnet 4.6正式发布，在前沿性能、编码能力和专业工作方面实现了全面突破。

  • Claude Partner Network - 3月12日，Anthropic宣布向Claude Partner Network投资1亿美元，这是继Claude生态之后的又一重大战略布局。

  • Anthropic Institute成立 - 3月11日，Anthropic宣布成立The Anthropic Institute，专注于AI安全和研究。

  • 收购Vercept - 2月25日，Anthropic收购了Vercept，以增强Claude的计算机使用能力。

  • Claude on Mars - 更令人兴奋的是，Claude成为首个协助NASA火星探测任务的AI，帮助Perseverance火星车行驶了400米。

LangChain：企业级Agent平台重磅发布

LangChain在Agent领域持续深耕，3月发布了多项重要更新：

核心产品

  1. 企业级Agentic AI平台 - LangChain与NVIDIA联合发布企业级Agent平台，提供从构建、部署到监控的全流程解决方案。

  2. Agent Harness理论 - LangChain提出了”Agent = Model + Harness”的创新理论，Harness工程成为Agent开发的核心方法论。

  3. GTM Agent成功案例 - LangChain分享了如何构建GTM Agent，实现了250%的潜在客户转化率提升，每位销售代表每月节省40小时。

  4. LangSmith CLI & Skills - 发布了全新的CLI工具和Skills系统，让AI编码代理（如Claude Code、Codex）能够更好地与LangChain生态系统集成。
